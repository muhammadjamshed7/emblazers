Quiz Creator Page (/curriculum/teacher/quizzes)
Quiz List View:
Table columns: Title | Class/Section | Subject | Questions | Marks | Time Slot | Status | Actions
Status badges:

Draft (gray) — not published
Scheduled (blue) — published but start time in future
Active (green) — currently within time slot
Expired (red) — past end time

Actions: Edit | View Attempts | Delete | Publish/Unpublish toggle
"Create New Quiz" button → opens Create Quiz page (/curriculum/teacher/quizzes/create)
Create Quiz — 3 step wizard:
Step 1: Quiz Settings

Quiz Title
Instructions for students
Class (from assignments)
Section (from assignments)
Subject (from assignments)
Time Limit (minutes) — number input with +/- buttons
Start Date & Time — date+time picker
End Date & Time — date+time picker
Total Marks (auto-calculated from questions, shown as read-only)
Passing Marks — number input

Step 2: Add Questions

"Add Question" button → inline question form appears:

Question Text (textarea)
Question Type: [ MCQ ] [ True/False ] [ Short Answer ]
If MCQ:

4 option inputs labeled A, B, C, D
Radio button next to each to mark correct answer


If True/False:

Toggle: Correct Answer is TRUE / FALSE


If Short Answer:

"Model Answer" textarea (for teacher reference only, not shown to students)


Marks for this question (number input)
"Add to Quiz" button


Questions list below with drag-to-reorder
Each question card shows: Q number, question preview, type badge, marks, edit/delete buttons
Running total marks shown at bottom

Step 3: Review & Publish

Full preview of quiz as student would see it
Summary: Total questions, Total marks, Time limit, Time slot
Two buttons: "Save as Draft" | "Save & Publish"
